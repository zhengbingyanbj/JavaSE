需求2：按要求进行查询

	1) 查询工资最高的员工是谁？
		提示：
		先写出查询最高工资的SQL ： select max(字段名) from 表名;
		将SQL进行嵌套：select * from 表名  where salary = ( select max(价格) from 表名);

	2) 查询工资小于平均工资的员工有哪些？
		2.1) 查询平均工资
				提示：select avg(字段名) from 表名;
				
		2.2) 小于平均工资的员工
				提示： 将SQL进行嵌套：select * from 表名  where salary < (select avg(字段名) from 表名);

	3) 查询大于5000的员工，来至于哪些部门，输出部门的名字
	
		3.1) 查询大于5000的员工，来至于哪些部门的id
			select 部门id from 表名 where salary >5000;

		3.2) 查询部门表，将上面的Sql作为查询条件，使用 in关键字，判断是否包含
			select d.* from dept d where d.id in  (select 部门id from 表名 where salary >5000;);

	4) 查询开发部与财务部所有的员工信息
		4.1) 子查询
			提示：先查询部门表中，开发部 和 财务部的 id。
			select d.id from 部门表 d where d.name='开发部' or d.name='财务部';
			
			提示：查询员工表,将上一条SQL 作为当前SQL的 条件
			select * from employee where dept_id in (select d.id from 部门表 d where d.name='开发部' or d.name='财务部');

		4.2) 表连接
			提示：使用 inner join 将 employee 和 dept记性连接，on 后面的条件是employee表的外键  dept_id = dept表的主键 id
		    select e.* from 员工表 e inner join 部门表 d on e.dept_id = d.id where d.name='开发部' or d.name='财务部';
