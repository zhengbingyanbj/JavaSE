需求1：SQL约束
	主键约束的概念：
		1) SQL语法中如何表示一个字段是主键？ primary(主键) key  
			
		2) 主键的作用是什么？ 用于表示数据库表中的数据唯一不重复
			
		3) 每个表可以有几个主键?  能有多个 ， 建议有一个主键
		
		4) 表中的主键特点是什么？ 主键必须填写 唯一性，
	
	如何添加主键约束(练习前两种添加方式)：
			方式一：创建表时，在字段描述处，声明指定字段为主键
				示例：CREATE TABLE p1(
						id int PRIMARY KEY,
						name varchar(200),
						idCard varchar(50)
					 );
				
			方式二：创建表时，在constraint约束区域，声明指定字段为主键
				示例：CREATE TABLE p2	(
						id int,
						name varchar(200),
						idCard varchar(50),
						PRIMARY KEY (id,name)
					  );
										
			方式三：创建表之后，通过修改表结构，声明指定字段为主键：（了解）
