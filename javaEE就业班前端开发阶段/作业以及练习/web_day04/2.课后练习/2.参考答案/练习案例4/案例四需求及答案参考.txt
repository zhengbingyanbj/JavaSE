需求解析：
	1.根据效果图可知，本题是要完成一个省市联动的案例
	2.在省市联动中我们知道省市这两个文本框实际是两个多选框，其中省的多选框中选项是固定的，而市的多选框呢要根据省的多选框所选的值的不同而改变
	3.所以说省市这两个多选框是有关系的，有什么关系呢其实就是我们学的二维数组的关系。二维数组中每个索引对应一个数组，而我们这可以理解为一个省对应多个市的数组。
	4.除了知道二者之间的关系，我们还要知道怎么串联这个关系，这里我们就要用到事件了，是不是我们只要一选择一个新的省就有对应的事在市的多选框中产生？所以这里我们就需要给省的多选框绑定一个改变事件，只要一触发事件我们就可以根据选的省去二维数组中找对应的市。
	5.找到对应的市，根据dom树我们将市添加进市的多选框中就可以了

			function selectCity(val){
				//1 获得全中的省value值
				
				//2 通过value获得对应的所有的市
				var allCity = arr[val];
				
				//3 遍历所有市，并将所有的市添加都select  -- cityId
				// 3.1 获得市select
				var citySelectObj = document.getElementById("cityId");
				// * 设置默认值
				citySelectObj.innerHTML = "<option value=''>----请-选-择-市----</option>";
				
				// 3.2 遍历所有市，创建市元素，并追加到市select中
				for(var i = 0 ; i < allCity.length ; i ++){
					// 创建 option元素
					var optionObj = document.createElement("option");
					// 设置 创建文本节点
					var textObj = document.createTextNode(allCity[i]);
					// 将文本追加到option中
					optionObj.appendChild(textObj);
					
					// 将 option 追加到select中
					citySelectObj.appendChild(optionObj);
				}
				
			}
